# Rate-My-Courses
⭐️ A website where Milton Academy students can login and review courses they've taken at Milton. This project was built by Bryan Sukidi '24 and Melvin Joseph '24.


## To start the React.js FRONTEND:
CD into the frontend folder and run:
``npm start``

## To start the Nest.js / MongoDB BACKEND:
CD into the backend folder and run:
``npm run start:dev``
